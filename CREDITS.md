# Credits

## Music

The web version uses downloaded classical recordings from Wikimedia Commons.

- `web/assets/music/goldberg_aria.ogg` - J. S. Bach, *Goldberg Variations: Aria*, Wikimedia Commons file `Goldberg_Variations_01_Aria.ogg`.
- `web/assets/music/gymnopedie_no1.ogg` - Erik Satie, *Gymnopedie No. 1*, Wikimedia Commons file `Gymnopedie_No._1..ogg`.
- `web/assets/music/toccata_bwv565.ogg` - J. S. Bach, *Toccata and Fugue in D minor, BWV 565*, Wikimedia Commons file `Toccata_et_Fugue_BWV565.ogg`.

These were selected as no-fee classical recordings suitable for static web hosting. Keep this file updated if tracks are replaced.
